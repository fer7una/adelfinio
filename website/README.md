# Website (legal pages)

Contenido web minimo para TikTok Developers:

- Home: `website/index.html`
- Privacy: `website/privacy-policy.html`
- Terms: `website/terms-of-service.html`

## Deploy rapido con GitHub Pages

1. Push del repo a GitHub.
2. En Settings -> Pages, usar branch `main` y carpeta `/website` (o publicar con GitHub Actions).
3. Anotar URLs publicas y cargarlas en TikTok Developers.
