# Roadmap

## Fase 0 - Foundation (actual)

- Infra local (Docker, n8n, Postgres, Redis, MinIO)
- Estructura de repo
- Schemas JSON y ejemplos
- Skeletons de workflows versionables
- Scripts de bootstrap, validacion y plan diario

## Fase 1 - Story Core

- Parser robusto de fuentes en `docs/chronicles/`
- Motor lineal cronologico de episodios
- Estado persistente de personajes con reglas de continuidad
- Guardrails narrativos (plot twist en principales)

## Fase 2 - Media Pipeline

- Integracion generacion visual/voz/subtitulos ES
- Render automatizado y empaquetado para TikTok
- Derivacion automatica de teaser por principal
- Control de calidad tecnico (duracion, bitrate, subtitulos)

## Fase 3 - Human Review + TikTok Ops

- UI/flujo de revision humana (aprobar/editar/regenerar parcial)
- Envio automatico a TikTok para revision manual
- Retries, alertas, metricas de coste y throughput (objetivo 4 videos/dia)
- Hardening operativo y runbooks
